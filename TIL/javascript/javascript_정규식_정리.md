# javascript) 정규식 정리



### 정규식 규칙

1. ###### 숫자 : [0-9]

2. ###### 공백 : \s

3. ###### 문자 : [a-zA-Z0-9] or \w

4. ###### 시작부분 : ^

5. ###### 끝부분 $

6. ###### 아닌것 :   ^  / 대문자 ex) [ ^s]  or \S => 공백이 아니다.

7. ###### ....  더 정리 해야함... 







### 사용방법

```javascript
정규식.test(테스트할_문자열); // 참거짓 반환

테스트할_문자열.replace(정규식, ""); // 정규식으로 수정
```





### 활용

```javascript
var number = /[^0-9]/g
var english = /[a-z]/ig
var special_character = /[`~!@@#$%^&*|₩₩₩'₩";:₩/?]/gi
var korean = /[a-z0-9]|[ \[\]{}()<>?|`~!@#$%^&*-_+=,.;:\"'\\]/g
var email = /^[-A-Za-z0-9_]+[-A-Za-z0-9_.]*[@]{1}[-A-Za-z0-9_]+[-A-Za-z0-9_.]*[.]{1}[A-Za-z]{1,5}$/

```
